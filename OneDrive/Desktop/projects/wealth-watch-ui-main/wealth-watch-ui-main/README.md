# Wealth Watch UI

A modern financial management dashboard built with React and TypeScript. Track your expenses, manage budgets, and gain insights into your spending habits with interactive charts and intuitive interfaces.

## Features

- **Dashboard Overview**: Get a quick snapshot of your financial status with key metrics and visualizations.
- **Transaction Management**: Add, view, and categorize your transactions easily.
- **Budget Tracking**: Set budgets for different categories and monitor your spending against them.
- **Expense Analytics**: Visualize your monthly expenses and spending insights with charts.
- **Category Breakdown**: See how your spending is distributed across different categories with pie charts.
- **Responsive Design**: Works seamlessly on desktop and mobile devices.

## Tech Stack

- **Frontend Framework**: React with TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **UI Components**: shadcn/ui
- **Charts**: Recharts (via shadcn/ui chart components)
- **Package Manager**: Bun
- **Linting**: ESLint
- **Configuration**: PostCSS, TypeScript configs

## Installation

1. Clone the repository:
   ```sh
   git clone <YOUR_GIT_URL>
   cd wealth-watch-ui-main
   ```

2. Install dependencies:
   ```sh
   bun install
   ```

3. Start the development server:
   ```sh
   bun run dev
   ```

4. Open your browser and navigate to `http://localhost:5173` (or the port shown in the terminal).

## Usage

- **Adding Transactions**: Use the Transaction Form to log your income and expenses.
- **Managing Budgets**: Set and adjust budgets for different categories in the Budget Management section.
- **Viewing Insights**: Check the Dashboard and Spending Insights for visual breakdowns of your finances.
- **Charts**: Explore interactive charts for budget comparisons, monthly expenses, and category distributions.
